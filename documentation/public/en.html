ESPHome Intex Pure Spa

Introduction

This project is a conversion of the excellent esp8266-intexsbh20 project to ESPHome for easier installation with Home Assistant. This firmware is only compatible with Intex SB-H20 Spa. If you wish to use your spa with another home automation system, refer to the initial project.

Requirements

  • Home Assistant
  • An ESP8266, the electronic board and connectors (more info here)
  • A micro USB cable
  • The password for your 2.4 Ghz Wi-Fi network
  • Google Chrome (or a Chromium-based browser like Microsoft Edge) on a computer (not Android/iOS)

Installation

Installation is done directly through the browser. Simply connect the ESP8266 to your computer using the micro USB cable and click the install button. Follow the instructions. You will need to enter your Wi-Fi password to connect the ESP8266 to your Wi-Fi. The spa will then be automatically discovered by Home Assistant as an ESPHome device.

Your browser is not compatible.

You can only launch the installation from a secure page (HTTPS).

Problèmes connus

  • The "Power" switch may not work at times.
  • The temperatures are equal to 65.535°C if the card is not connected to the spa.
