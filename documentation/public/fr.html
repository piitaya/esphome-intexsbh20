ESPHome Intex Pure Spa

Introduction

Cet projet est une conversion de l'excellent projet esp8266-intexsbh20 vers ESPHome pour une installation plus simple avec Home Assistant. Ce firmware n'est compatible qu'avec le spa Intex SB-H20. Si vous souhaitez utiliser votre spa avec un autre système domotique que Home Assistant, reportez-vous au projet unitial.

Prérequis

  • Home Assistant
  • Un ESP8266 ainsi que la carte électronique et les connecteurs (plus d'info ici)
  • Un câble micro USB
  • Le mot de passe de votre réseau Wi-Fi 2.4 Ghz
  • Google Chrome (ou un navigateur basé sur Chromium comme Microsoft Edge) sur un ordinateur (pas Android/iOS)

Installation

L'installation se fait directement via le navigateur. Il suffit de brancher l'ESP8266 à votre ordinateur à l'aide du câble micro USB et de cliquer sur installer. Suivez les instructions. Vous serez invitez à renseigner votre mot de passe Wi-Fi pour connecter l'ESP8266 à votre Wi-Fi. Le spa sera alors automatiquement découvert par Home Assistant en tant qu'appareil ESPHome.

Votre navigateur n'est pas compatible.

Vous ne pouvez lancer l'installation que depuis une page sécurisée (HTTPS).

Problèmes connus

  • L'interrupteur "Power" peut ne pas fonctionner par moment.
